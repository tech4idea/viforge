viwork Agent Harness Data Model

短期核心对象关系：变更对象、agent 分层配置、记忆策略、知识检索、输入快照、评估案例、人工评分和发布状态。

Change Object

  • • AgentSpec carries prompt, skill, memory, retrieval, tool and model refs
  • • AgentLayerConfig records system/specialist boundaries
  • • Active status is controlled by release decisions

Repro Objects

  • • RunInputSnapshot captures full project lightweight state
  • • EvalFixture fixes memory records and knowledge hits
  • • WorkspaceManifest suggests structural assertions

Evaluation Objects

  • • EvalRun stores tool events, output and file diff
  • • Assertions cover hard program checks
  • • HumanReviewRubric covers creative quality
